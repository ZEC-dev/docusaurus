**这是AI的基础**  
Sigmoid 激活函数
    公式: `$f(x) = 1 / (1 + e^{-x})$`
    输出范围: (0, 1)  
```python
def sigmoid(x):
    """
    Sigmoid 激活函数
    公式: f(x) = 1 / (1 + e^(-x))
    输出范围: (0, 1)
    """
    return 1 / (1 + np.exp(-x))

def sigmoid_derivative(x):
    """Sigmoid 函数的导数"""
    s = sigmoid(x)
    return s * (1 - s)
```
